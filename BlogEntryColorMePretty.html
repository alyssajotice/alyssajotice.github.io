Color Me Pretty

Fall 2018

For my iOS Development capstone, I built an application for iPads that imitated a coloring book for children. I wanted the UI to be pleasant, easy to use, and feature rich. Pictured beneath are the screenshots of the different pages in the app. I utilized Swift libraries and frameworks to implement the following features for users: tutorial, free draw, eraser, graded color by number, color with a friend, picking colors and line widths, save image to camera roll, clear image, and a library of images to color.

As with any programming project, some features were easy to implement while others were much more difficult. The first technical challenge I had was supporting the multiplayer mode in the game. I needed two iPads to be able to communicate almost instantly so that two people could color a picture together at the same time. I ended up finding the Apple MultipeerConnectivity framework to stream the data from one device to the other. I sent the data points from one user's drawings to the other device and rebuilt the Bezier path that made up the actual line. This part of the project took a lot of trouble shooting but ended up being really interesting to implement.

The other technical hurdle in this project was the grading of the numbered pictures. The original goal of the app was to be a color by number type of game, where the user put numbered colors into numbered sections of the picture. My app would then grade the percentage of the photo that was correctly colored. Finding an efficient way to do this was challenging, since the screen is quite large and there are many points that need to be kept track of and sorted through to make sure that the user is coloring the right color in the right area. I ended up having to only use photos with a smaller amount of area so that it would be usable. In the future, the process would need to be multithreaded so that every single pixel can be checked in a reasonable amount of time for the user. At the end of the semester, the grading feature was about 80% effective according to my estimates. This project is one of the technical achievements that I am most proud of, plus it was incredibly fun to work on. It sparked my interest in iOS development and I can see myself doing more of it in the future. Pictured beneath are the screenshots of the different pages in the app.
