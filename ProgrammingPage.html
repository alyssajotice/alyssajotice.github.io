Synthesizer Control Board

One of the classes I took during my freshman year was a class about musical synthesizer programming, using the programming language SuperCollider. The class gave me a lot of creative freedom to make whatever I desired for the final project. I used a Korg NanoKontroll 2 and a MIDI keyboard to build and program a sound board with changeable synthesizers.

The NanoKontroll 2 board had 9 control sections that input MIDI data to the computer, so I programmed each one to play a different synth and control specific characteristics of that synth. This project was really enjoyable for me becaue I got to combine two things that interest me, music and programming! You can read more about how I executed this project in the blog post, which is here!
