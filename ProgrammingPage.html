Sofware Engineering Intern
Capital One

During the summer of 2019, I interned at Capital One Financial in their Technical Development Program Internship in McLean Virginia. I was placed on a team that worked on a customer notification platform. I was given ownership of my own microservice to help expand the functionality of the platform. The goal of the service was to allow a segment of customers to be sent a notification email.

I was in charge of designing and building the back end of the project, while another intern built out the front end. My project was a Java-Maven-Spring application that pulled segment data from AWS S3. The file taken from AWS was then processed, encrypted/encoded, and an API was called for each piece of data. I used the Cassandra NoSQL database to get template information from the front end. I designed my own error reporting method and kept usage statistics and reports that were sent via AWS SNS and uploaded to S3.

My project had a high business value, and the goal of the entire summer was to put it into production by the end of my internship. I had never seen real production testing, and I got to put my code through local, system, integration, and performance testing. I learned how code is configured for these different environments and used a Jenkins job to do builds. I wrote a release plan and got to run point on the release. Pretty much every single technology used in the project was something I had never seen before, and I gained invaluable experience from doing the end to end production of a real product used by the company. This was the greatest learning experience of my career thus far and I am looking forward to using my skills for my future career.

Color Me Pretty

During fall 2018, I took the iOS Application Development elective for my major. I learned a lot about Swift, and got to do a Capstone project for the class. My roommate suggested a coloring app for the iPad, and I decided to go with the idea. I created a children-focused app that allows the user to color different images.

Swift has been my favorite elective so far, so I was really passionate about my project and wanted to add a lot of features that would allow me to learn as much as possible about Swift. I included a tutorial, free draw, graded color by number, save image, pair coloring with another device, and different images to color.

I wanted the UI to be very pleasant, so I really focused on design. To see screenshots, check out my blog post about the project here!

Synthesizer Control Board

One of the classes I took during my freshman year was a class about musical synthesizer programming, using the programming language SuperCollider. The class gave me a lot of creative freedom to make whatever I desired for the final project. I used a Korg NanoKontroll 2 and a MIDI keyboard to build and program a sound board with changeable synthesizers.

The NanoKontroll 2 board had nine control sections that input MIDI data to the computer, so I programmed each one to play a different synth and control specific characteristics of that synth. This project was really enjoyable for me becaue I got to combine two things that interest me, music and programming! You can read more about how I executed this project in the blog post, which is here!
